# 成为一名靠谱的前端

## 背景

作为一个程序员出身的创业者，深刻地体会了寻找前端的痛苦与艰难。

于是萌发了**一个想法**：为什么不让自己成为一名合格的前端呢？

说干就干，于是开始了为期两个月的前端探索。

## 规划
**9月、10月**

- 基础学习：10天
- html/html5实战：7天
- css实战：7天
- JS实战：14天
- 小型项目实战（待规划）：14天
- App实战：8天
